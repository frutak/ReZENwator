# ReZENwator
Otwarty CMS i Channel Manager dla najmu krótkoterminowego (Self-Hosted).

🔄 Dwukierunkowa synchronizacja: Booking.com, Airbnb, Slowhop, AlohaCamp.

🧘‍♂️ Zautomatyzowany Zen: System na bieżąco śledzi statusy rezerwacji i księgowanie płatności na Twoim koncie, zwalniając Cię z ręcznej papierologii.

🛒 Direct Booking Engine: Gotowy portal do przyjmowania rezerwacji bezpośrednich.

💸 100% darmowy: Tworzony przez hosta dla hostów. Zbuduj swój niezależny biznes bez płacenia abonamentów komercyjnym platformom.

Każdy z tych wariantów jasno pokazuje, że znasz realia polskiego i europejskiego rynku (Slowhop, AlohaCamp), a jednocześnie rozwiązujesz bardzo konkretny, bolesny problem (papierologia, płatności, synchronizacja kalendarzy), co w świecie open-source jest najlepszym magnesem na użytkowników.

---

### Features
*   **Unified Calendar**: Visualizes bookings from all channels (Slowhop, Airbnb, Booking.com, Alohacamp).
*   **Automated Enrichment**: Parses emails from portals to enrich iCal data with guest names, emails, phones, and pricing.
*   **Payment Matching**: Scrapes bank transfer emails to automatically match payments to bookings using fuzzy logic.
*   **Pricing Auditor**: Scrapes portals daily to ensure your prices are consistent across all channels.
*   **Guest Portal**: Built-in simple website for direct bookings and guest communication.
*   **Daily Alerts**: Sends a summary of check-ins, check-outs, and pending payments to the host.
*   **Double-Booking Detector**: Alerts you immediately if overlapping bookings are detected across channels.
